1.9.Формы.Знакомство.

Первая Форма.

Формы нужны для того, чтобы отправлять данные с веб-страницы на веб-сервер, который сможет эти данные обработать: зарегистрировать пользователя, создать сообщение на форуме, отправить письмо и так далее.
<form> имеет два атрибута action & method.
action - атрибут, который задает ссылку, куда отправлять данные из формы.
method - атрибут, показывающий способ отправки данных.

У method есть два способа отправки - get & post.

get - лучше использовать в поисковых формах, потому что он позволяет получить ссылку на результаты поиска и передать её кому-то.

post - посылает данные в теле HTTP-запроса и используется, когда нужно отправить много данных и ссылка на результат обработки этих данных не нужна. Например, при редактировании личного профиля.



Текстовое поле ввода.

<input> - одиночный тег, в котром создается форма.
Имеет два атрибута: type & name.

type - отвечает за то, какого типа форма будет отправляться: текст, пароль или можно смотреть здесь.
name - задает специальное имя для формы. ИМЯ А НЕ АЙДИШНИК.

Атрибут id является идентификатором элемента.


Атрибут value по умолчанию задает значение полю ввода.



<label>

<label> - тег, который связывает форму и инпут логически.


Тег имеет атрибут for, в котором указывается айдишник инпута. Таким образов поле ввода и лейбл связываются логически


Пример.


                <form action="#">
                    <label for="example2">
                        <input type="text" name="input" id="example2">
                    </label>
                </form>

Большое окно поля ввода.

<textarea> - большое окно поля ввода.


Этот тег имеет 4 основных атрибута: name, id, rows, cols.


Первые два атрибута идентичны инпуту, а вот rows - задает высоту поля, cols - задает ширину поля.


value тут нет, поэтому занчение по дефолту ставится между <textarea></textarea>>



Кнопка Чекбокс.

У инпута в тайпе также есть свойство checkbox.

Также у этого checkbox есть свойство checked. Оно делает галочку внутри по дефолту отмеченной.

Поле для загрузки файлов.

Используется инпут с типом file.
Для того, чтобы этот тип работал, в форме необходимо указать enctype со значением multipart/form-data.

Этот атрибут указывает браузеру, в каком виде пересылать данные. Если вы хотите отправить на сервер файл, данные из формы эффективнее всего передавать по частям. За это и отвечает значение multipart/form-data
